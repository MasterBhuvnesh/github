# Git

Git is a distributed version control system that allows you to track changes, collaborate with others, and manage code efficiently.  
Before using it, you need to **install Git** and **configure basic settings**.

---

## Installing Git

### On macOS

1. Download the official installer: [Git for macOS](https://git-scm.com/download/mac)
2. Follow the installation prompts.
3. Verify installation:

```bash
git --version
```

---

### On Windows

1. Download the installer: [Git for Windows](https://git-scm.com/download/win)
2. Run the installer and follow the steps (default options are fine).
3. Confirm installation:

```bash
git --version
```

---

### On Linux (Debian/Ubuntu)

Run the following command:

```bash
sudo apt-get update
sudo apt-get install git
```

Verify installation:

```bash
git --version
```

---

## Setting Up Git

Once installed, configure your identity so Git knows who is making changes:

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

To check your configuration:

```bash
git config --list
```

---

## Example

```bash
$ git config --global user.name "Bhuvnesh Verma"
$ git config --global user.email bhuvnesh@example.com
```
